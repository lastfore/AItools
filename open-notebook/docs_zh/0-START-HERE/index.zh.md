## 你能做什么？

- 📄 **上传内容**：PDF、网页链接、音频、视频、文本
- 🤖 **与 AI 对话**：就文档提问并获得引用
- 📝 **生成笔记**：AI 创建摘要与洞察
- 🎙️ **创建播客**：将研究转化为专业音频内容
- 🔍 **搜索**：全文与语义搜索全部内容
- ⚙️ **转换**：提取洞察、分析主题、创建摘要

## 为什么选择 Open Notebook？

| 功能 | Open Notebook | Notebook LM |
|---------|---|---|
| **隐私** | 自托管，你掌控 | 云端，Google 服务器 |
| **AI 选择** | 17+ 提供商 | 仅 Google 模型 |
| **播客说话人** | 1–4 人可定制 | 仅 2 人 |
| **成本** | 完全免费 | 免费（但你的数据） |
| **离线** | 支持 | 不支持 |

## 前置条件

- **Docker**：所有路径均使用 Docker（免费）
- **AI 提供商**：云端 API 密钥，或使用免费本地模型（Ollama）
